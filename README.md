# BMW Sales Analysis Dashboard

## Project Overview
This project contains an interactive Power BI dashboard designed to analyze and visualize BMW car sales data. It provides in-depth insights into sales performance, geographical distribution, and various BMW car models.

## Directory Structure
- **Dashboard/**: Contains the main Power BI dashboard file (`BMW DASHBOARD.pbix`).
- **Dataset/**: Contains the raw data sources used for the analysis.
  - `BMW_Sales_Data.xlsx`: Core dataset with BMW sales figures and transactions.
  - `Car Images.xlsx`: Dataset containing URLs or references to corresponding BMW car model images for visual enhancement.
  - `Countries with Flags URL.xlsx`: Dataset with image URLs used for mapping country flags.
- **ScreenShots/**: Contains visual previews of the dashboard pages.
  - `BMW DASHBOARD_page-0001.jpg`
  - `BMW DASHBOARD_page-0002.jpg`
  - `BMW DASHBOARD_page-0003.jpg`

## How to Use
1. Ensure you have **Microsoft Power BI Desktop** installed on your machine.
2. Download or clone this repository to your local system.
3. Navigate to the `Dashboard` folder and open the `BMW DASHBOARD.pbix` file.
4. If the visuals do not load, you may need to refresh the data connection by repointing the data sources to the downloaded Excel files located in the `Dataset` directory.
5. Interact with the dashboard's filters and slicers to explore the BMW sales data!

## Key Features
- **Interactive Visualizations**: Dynamic charts and graphs for exploring sales revenue, volume, and trends over time.
- **Geographic Analysis**: Sales breakdowns by country, augmented with flag visuals for easy recognition.
- **Model-Specific Insights**: Detailed performance tracking of different BMW car models with integrated car imagery.

## Tools Used
- **Microsoft Power BI**: For data modeling, visualization, and dashboard creation.
- **Microsoft Excel**: For dataset storage and management.
